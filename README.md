# cs_assault_go

This map is made for counter strike 1.6

# Used tools:
Valve Hammer Editor 3.5
ZHLT 3.0

# How to compile:
-copy & paste "source folder to your ZHLT Folder"
-then run cs_assault_go.bat

# Editing map:
-open cs_assault_go.rmf
